# Resumo Executivo
- Repositório do desafio técnico Fase 4 do curso “IA para Devs” (GitHub: [bodjay/ia-para-devs](https://github.com/bodjay/ia-para-devs)). Linguagens usadas incluem Python e TypeScript.

## Como Executar
- `git clone https://github.com/bodjay/ia-para-devs.git` e acesse `fase-4/tech-challenge-typescript`.  
- Execute `npm install` para instalar dependências.  
- Comando de inicialização: `start.sh`
- **Node.js (versão):** >20.18.0
